# C Basics

This started out a practice session on trying to create a LinkedList from scratch in C. I was studying for an interview and new this would be a good way to do just that.

However, it has evolved into a series of basic data structures, memory management, and algorithm concepts that are covered in most software engineering degrees.
Although, I don't remember ever being forced to go this deep on them, which is likely partly why I didn't remember all the specifics over a decade later.

The structures are good to reference, but the most important thing is to document the major lessons. Checkout lessons.c for a summary of the lessons.
